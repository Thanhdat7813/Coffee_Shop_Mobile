## 🎯 Mô tả bài tập
Ứng dụng **Coffee Shop Mobile App** được xây dựng theo thiết kế Figma, gồm 3 màn hình chính:
- **Onboarding:** giới thiệu ứng dụng và điều hướng vào trang chủ  
- **Home:** hiển thị danh sách sản phẩm, tìm kiếm và lọc theo danh mục  
- **Detail Item:** xem chi tiết sản phẩm, chọn size và hiển thị giá  

Dữ liệu sản phẩm được tải từ file **data.json**.
